Déployer ses projets gratuitement avec Vercel en 5 minutes

Le guide rapide que j'aurais voulu avoir quand j'ai mis mon premier site en ligne. Zéro configuration, zéro serveur, zéro coût.

Rayan Remili
15 novembre 2024
3 min de lecture

Tous mes projets — NextStage, Yacine Visa, Quotiva, ce portfolio — sont hébergés sur Vercel. Gratuitement. En quelques minutes chacun. Si tu fais du web et que tu n'utilises pas encore Vercel, cet article est pour toi.

⚡ Vercel est gratuit pour les projets personnels. Pas de carte bancaire requise pour commencer.

Les 5 étapes pour mettre ton site en ligne

1
Crée un compte GitHub
Si tu n'en as pas encore, va sur github.com et crée un compte. GitHub va stocker ton code et servir de pont avec Vercel. C'est indispensable.
2
Crée un repository et pousse ton code
Sur GitHub, clique "New repository", donne-lui un nom, et pousse tes fichiers dedans. Si tu n'as jamais utilisé Git, les commandes essentielles sont :
git init git add . git commit -m "premier commit" git remote add origin https://github.com/tonnom/tonrepo.git git push -u origin main
3
Va sur vercel.com et connecte GitHub
Crée un compte sur vercel.com en te connectant avec GitHub. Vercel va demander l'accès à tes repositories — accepte.
4
Importe ton projet
Clique "Add New Project", sélectionne ton repository GitHub. Pour un site HTML/CSS/JS simple, Vercel détecte tout automatiquement. Clique "Deploy".
5
Ton site est en ligne ✓
En 30 secondes, Vercel te génère une URL du type monprojet.vercel.app. C'est ton site, accessible partout dans le monde, gratuitement.

Le meilleur : les mises à jour automatiques

C'est là où Vercel devient vraiment magique. À partir de maintenant, chaque fois que tu fais un git push, Vercel redéploie automatiquement ton site. En 30 secondes, les changements sont en ligne. Zéro intervention manuelle.

# Tu modifies ton code, puis : git add . git commit -m "correction du bug de navigation" git push # 30 secondes plus tard → site mis à jour

Domaine personnalisé

Si tu veux une vraie URL comme rayan-remili.com au lieu de rayan-remili.vercel.app, tu peux connecter un nom de domaine dans les paramètres du projet. Les domaines coûtent environ 10-15€/an chez des registrars comme Namecheap ou OVH. La configuration dans Vercel prend 5 minutes.

Ce que Vercel offre gratuitement

  • Hébergement illimité pour les projets personnels
  • HTTPS automatique — ton site est sécurisé par défaut
  • CDN mondial — ton site se charge vite partout
  • Déploiements automatiques sur chaque push
  • Previews pour chaque branche Git

Il n'y a vraiment aucune raison de ne pas utiliser Vercel pour ses projets personnels. C'est l'outil que j'aurais voulu connaître dès le début.
